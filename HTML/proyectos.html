Aplicación para Acceso a una Base de Datos

Este proyecto consiste en una aplicación que permite acceder a una base de datos para gestionar información de manera eficiente. SQL para la gestión del backend y la base de datos. Es una herramienta útil para aprender sobre la interacción entre aplicaciones y bases de datos, así como para desarrollar habilidades en programación y diseño de interfaces.

Juego: Tres en Raya

Este proyecto consiste en un juego clásico de tres en raya, donde los jugadores pueden competir entre sí. Se utiliza HTML, CSS y Java para crear una interfaz interactiva y dinámica. Es una excelente manera de aprender sobre la lógica de programación y la creación de juegos simples.

Pagina Web

Este proyecto fue un trabajo final de prácticas en Deloitte, el cual consistía en desarrollar una página web utilizando HTML, CSS y JavaScript. El objetivo era crear una página web interactiva y visualmente atractiva que pudiera servir como portafolio personal.
